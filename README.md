### 🐬 무엇을 하는가
 - 파이썬 공부, 클론 코딩

### 🦈어떻게 하는가
 1. 주어진 과제 수행
 2. 작성한 코드를 관련 폴더에 저장
 3. 저장한 파일을 개인별 fork한 repo의 main으로 push
 4. 개인별 깃허브 repo에 push된 main branch를 스터디 저장소의 main branch로 pr을 보낸다
 5. 반복

### 🐬주의사항
1. 코드 작업 전과 후에 최신화 시키기
2. 다른 사람의 폴더를 절대로 삭제하지 않기
3. pull은 원본 레포지토리를, push는 fork한 개인 레포지토리를 이용하기
